# My Curriculum Vitae

A landing page for a personal portfolio site, showcasing my skills, experiences, and projects.

## Features

*   **Organize Sections:** Includes sections for main information, experiences, skills, and projects.
*   **Responsive Design:** This was designed to be responsive and should work on all devices.
*   **CV JSON Data:** The curriculum vitae data is stored from `mycvdata.json` file, for easy update.

## Tech Stack

*   **Astro:** The web framework for building the website.
*   **TypeScript:** For type safety and improved developer experience.
*   **Tailwind CSS:** For styling the website.
*   **React:** Used for some interactive components.
*   **Vercel:** For deployment.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js (v22.x or later)
*   npm

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/seanvizm/astro-mycv
    ```
2.  Install PNPM via npm
    ```sh
    npm install -g pnpm
    ```
3.  Or install PNPM via brew
    ```sh
    brew install pnpm
    ```
4.  Install PNPM packages
    ```sh
    pnpm install
    ```

### Running the Development Server

To run the development server, use the following command:

```sh
pnpm run dev
```

This will start the development server at `http://localhost:4321`.
